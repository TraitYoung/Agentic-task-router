Axiodrasil 是一款基于第一性原理设计的 AI-Native 个人认知管理系统。它不只是一个聊天机器人，而是一个具备**多智能体协作 (Multi-Agent)**、**状态机保护 (State Machine)**与**宪法级协议 (Pydantic Protocols)**的认知架构，旨在高强度任务场景下平衡人类执行效率与身心承载力。



## 🛠️ 核心架构特性

* **多智能体动态路由 (Dynamic Routing)**: 基于 LangGraph 构建有向无环图 (DAG)，设计「逻辑审计」、「策略规划」、「情绪支持」、「医疗熔断」四大核心 Agent。
* **宪法级 Pydantic 协议**: 通过 Pydantic 定义 12 项系统交互协议，将模糊的自然语言强制转化为确定性的结构化数据，彻底消除模型路由幻觉。
* **物理级熔断机制 (Meltdown Protection)**: 内置身心状态机，根据时间戳与用户生理指标 (Pain Level) 自动触发强制休息逻辑，防止过度劳损。
* **自然语言驱动开发 (NL-Driven)**: 探索“架构设计 -> 协议定义 -> 指令注入”的全新开发范式。

## 🚀 快速点火 (Quick Start)

1.  **环境配置**:
    ```bash
    pip install langchain-openai langgraph pydantic python-dotenv
    ```
2.  **密钥设置**: 创建 `.env` 文件并填入你的 `QWEN_API_KEY`。
3.  **运行测试**:
    ```bash
    python test.py
    ```
